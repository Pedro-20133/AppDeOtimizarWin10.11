# App para Otimizar Win 10 e 11

Este é um app focado em otimizar o Win 10 e 11, algumas funções podem funcionar em versões anteriores(Ex: Win 7) mas o foca é no Win 10 e 11

O Software é 100% gratuito, sem vírus e utíl
